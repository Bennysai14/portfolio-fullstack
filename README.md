# Full-Stack Portfolio – React + Java + MySQL

This repository contains my personal developer portfolio, built as a full‑stack application with a React frontend and a Java/Tomcat + MySQL backend.

## Tech Stack

- Frontend: React, CSS, JSX (VS Code)
- Backend: Java, Tomcat (Eclipse), REST API
- Database: MySQL
- Build & Tools: npm, Git, GitHub

## Features

- Modern single‑page portfolio with sections:
  - Hero, About, Skills, Projects, Experience, Contact
- React contact form that sends data to a Java REST API running on Tomcat
- MySQL persistence for contact messages
- Clean, mobile‑responsive UI tailored for recruiter viewing

## Project Structure
portfolio-fullstack/
frontend/ # React portfolio (VS Code)
backend/ # Java API (Eclipse/Tomcat)

## How to Run Locally

### 1. Frontend (React)

cd frontend
npm install
npm start



Runs on: http://localhost:3000

### 2. Backend (Java/Tomcat)

- Open `backend` in Eclipse  
- Start the configured Tomcat server  
- Ensure the contact API endpoint is reachable at your configured URL (for example `http://localhost:8081/api/contact`)
